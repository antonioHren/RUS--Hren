# Wokwi Simulation Results

This folder contains waveform data exported from Wokwi simulations.

The file `wokwi-logic.vcd` is a Value Change Dump (VCD) file generated by Wokwi.

## Signals included

From the VCD file, the following signals are present:

- `D0` (wire 1 bit)
- `D1` (wire 1 bit)
- `D2` (wire 1 bit)

These signals represent digital outputs observed during the simulation.

## Time information

- Timescale: 1 ns
- The file contains timestamped signal transitions (e.g. `#0`, `#515338183`, etc.)
- Each timestamp indicates when a signal value changed

## Example interpretation

Example snippet:
#0
1!
#0
0"
#0
0#
#515338183
0!

This means:
- "!" sign represents a button interrupt, the """ sign represents UART interrupt and "#" represents the timer interrupt
- At time 0 ns, signal `D0` was HIGH
- Other signals were LOW
- At later timestamps, signals change state, indicating events in the simulation
